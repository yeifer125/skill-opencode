# My OpenCode Skills
Personal collection of skills for Opencode.
## Skills Included
- **openclaw-noauth-fix**: Apply no-auth fix for using free AI providers like kilo.ai without API key
## Usage
Each skill is in its own directory with a `SKILL.md` file containing instructions.
To use a skill:
1. Clone this repo
2. Copy the skill directory to `~/.openclaw/skills/`
3. The skill will be automatically loaded by OpenClaw
Or install globally with:
```bash
openclaw skills install <skill-name>
```
## Contributing
Add your skills as subdirectories with a `SKILL.md` file following the OpenClaw skill format.
